07.bună For循环语句

bună For循环语句

bună for循环可以遍历任何序列的项目，如一个列表或者一个字符串。

语法:

for循环的语法格式如下:

for (var i = 0; i <10; i++) {
    print i;
}

流程图:

for (var i=0;i<3;i=i+1){ //第⼀个实例
    print i;
}

运行实例>>

输出结果为：

0
1
2
