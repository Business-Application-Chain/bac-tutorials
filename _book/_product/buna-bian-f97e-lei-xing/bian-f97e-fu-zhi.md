# 变量赋值

Buna中的变量赋值需要使⽤用var关键字进行声明。

每个变量在内存中创建，都包括变量的标识，名称和数据这些信息。

每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。

等号\(=\)用来给变量赋值。

等号\(=\)运算符左边是一个变量名,等号\(=\)运算符右边是存储在变量中的值。例如:

```
var miles = 1000.0; // number
var name = "John"; // string
print miles;
print name;
```
