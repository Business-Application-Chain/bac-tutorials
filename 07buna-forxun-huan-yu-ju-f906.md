# Buna For循环语句

Buna for循环可以遍历任何序列的项目，如一个列表或者一个字符串。

**语法:**

for循环的语法格式如下:

```
for (var I = 0; I <10; I++) {:
    statements(s)
}
```

**流程图:**

<img src="./assets/for.png" alt = "for" title = "for-png" width = "400" height="400"/>

```
for (var i=0;i<3;i=i+1){ //第⼀个实例
    print i;
}
```
